## Project Description

This project was created to practice the basic Git commands from Lesson 2 of the Git & GitHub Bootcamp. The goal is to understand how Git tracks changes, stages files, creates commits, and stores project history.

## Learning Objectives

- Understand what a Git repository is.
- Learn the three Git areas:
  - Working Directory
  - Staging Area
  - Repository
- Practice creating commits.
- Learn how to use Git history.
- Build confidence with the Git workflow.

## Git Commands Practiced

```bash
git init
git status
git diff
git add
git commit -m "Commit message"
git log
git log --oneline
```

## Git Workflow

1. Create or edit a file.
2. Check the status.
3. Review changes.
4. Stage changes.
5. Commit changes.
6. View commit history.
7. Repeat.

## Example Workflow

```bash
git status
git diff
git add .
git status
git commit -m "Add project files"
git log --oneline
```
